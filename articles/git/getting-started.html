Getting Started with Git


What is Git and why should I use it?

Git is a decentralized version control system, a tool to help groups or indiviuals manage changes to a project over time.

Git is important to use because it provides a full history of all changes made to a project over its lifetime and allows you to rollback to older states.

Version control systems and more specifically Git, are used in almost every professional development environment so it is a super important tool to know how to use before you join one of these workplaces.

The links in the sidebar will get you started with the basics of Git and provide references for further learning.

Installing Git

  • Linux: most linux distros come with Git installed and it can be invoked through the git command. If your distro does not have Git installed you can install it using one of these commands:
    • Debian Based (Ubuntu, Mint, anything using apt package manager): apt-get install git
    • Arch Based (Arch, Antergos etc..): pacman -S git
    • Fedora: yum install git
  • Windows: download installer here
  • macOS: git should be preinstalled, invoke it with git in Terminal.
