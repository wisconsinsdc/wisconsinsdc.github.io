Basic Git Commands


Create Local Repository

Create a fresh repository:

git init [name]

Clone a remote repository (from GitHub or other):

git clone [project url]

Adding/Removing

Stage a file for the next commit:

git add [filename]

Stage all modified files:

git add .

Commit all staged files:

git commit -m "describe your changes"

Unstage a file:

git reset [filename]

Reset everything back to the last commit (warning: destructive):

git reset --hard

Branching

List local branches:

git branch

List local and remote branches:

git branch -av

Switch to a branch:

git checkout [branch_name]

Create a new branch:

git branch [new_branch_name]

Delete a branch:

git branch -d [branch_name]

Merge branch1 into branch2:

git checkout branch2 git merge branch1

Tag the current commit:

git tag [my tag]

Observing

List new/modified and stages files that haven't been committed:

git status

Show changes to unstaged files:

git diff

Show changes to staged files:

git diff --cached

Show all staged and unstaged file changes:

git diff HEAD

Show changes between two commits:

git diff [commit1] [commit2]

Show authors and change history for a file:

git blame [filename]

Show full change history:

git log
